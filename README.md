Online Resume Builder

##Project Overview
Online Resume Builder is a web-based application that helps users create professional resumes easily. Users can enter their personal details, education, skills, and experience, and generate a well-structured resume instantly.

##Features
User-friendly interface
Add personal information
Add education details
Add skills and experience
Generate professional resume
Simple and responsive design

##Technologies Used
HTML
CSS
JavaScript
